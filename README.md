# XYZ-Bookstore-Page
This is a simple HTML project that represents a basic Bookstore web page. It displays books with descriptions and includes buttons for user interaction like purchasing, viewing cart, and checkout.
This is a simple HTML project that represents a basic Bookstore web page. It displays books with descriptions and includes buttons for user interaction like purchasing, viewing cart, and checkout.

 Features:-
 Displays book titles and descriptions
 "Buy Now" button for each book
 View Cart and Checkout buttons
Organized using div containers
Beginner-friendly HTML structure
Technologies Used
HTML5
Code Overview

The page includes:

A main heading (XYZ Bookstore)
A short introduction paragraph
Two book cards:
Sally's SciFi Adventure
Dave's Cooking Adventure
Action buttons:
Buy Now
View Cart
Checkout
